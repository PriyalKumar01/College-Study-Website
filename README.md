## 🚀 Overview

**College Study Web** (Unlocked Nexus) is a next-generation academic platform designed to democratize education. It serves as a unified hub for students to access high-quality notes, calculate CGPA, track academic progress, and contribute to a growing repository of knowledge.

With a focus on **premium UI/UX**, accessibility, and community engagement, the platform empowers students to excel in their academic journey.

## ✨ Key Features

- **📚 Extensive Note Repository**: Access curated notes from top contributors across various branches and semesters.
- **🏆 Contributor Hall of Fame**: A gamified leaderboard recognizing top students with badges (Gold, Silver, Bronze) and distinct "Podium" styling.
- **📊 Advanced CGPA Calculator**: A comprehensive tool for calculating SGPA and CGPA with precision.
- **🔐 Secure Authentication**: Robust user authentication system powered by Supabase.
- **👤 Student Profiles**: Personalized dashboards to track contributions, coins, and academic stats.
- **🌗 Dark Mode Support**: A fully responsive interface with toggleable light and dark themes for comfortable reading.
- **📱 Mobile-First Design**: Optimized for seamless experience across all devices.

## 🛠️ Tech Stack

This project is built using modern web technologies to ensure performance, scalability, and maintainability.

- **Frontend**: [React](https://reactjs.org/) (v18) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To set up the project locally for development purposes, follow these steps.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  **Clone the repository** (Private Repo):
    ```bash
    git clone https://github.com/your-username/studyhub-unlocked-nexus.git
    cd studyhub-unlocked-nexus
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Environment Setup**:
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:8080`.
